# Brand Assets

This directory contains brand assets for the Electric Ireland Insights integration,
prepared for future submission to the [home-assistant/brands](https://github.com/home-assistant/brands) repository.

## Required files for core submission

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 256×256 px | Integration icon (square, transparent background) |
| `icon@2x.png` | 512×512 px | High-DPI icon |
| `logo.png` | Variable | Full Electric Ireland logo |

## Sourcing assets

Official Electric Ireland brand assets should be sourced from:
- Electric Ireland's official website: https://www.electricireland.ie
- Electric Ireland brand guidelines (contact their communications team)

The icon should use the Electric Ireland "EI" monogram or the flame logo on a white/transparent background.

## Submission process

When submitting to HA core:
1. Fork https://github.com/home-assistant/brands
2. Add assets to `custom_integrations/electric_ireland_insights/`
3. Open a PR following the brands repository contribution guidelines
